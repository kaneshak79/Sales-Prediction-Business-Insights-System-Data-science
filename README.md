# 📊 Sales Data Analysis Dashboard

## 🚀 Project Overview
This project is a Data Science (EDA) + Web Dashboard built using Python and Streamlit.  
It analyzes sales data to extract business insights such as category-wise sales performance and profit distribution.

---

## 🎯 Objective
To analyze sales data and answer key business questions like:

- Which product category has the highest sales?
- Which category generates the most profit?
- What are the key business insights from the dataset?

---

## 🛠️ Tech Stack

- Python 🐍
- Pandas 📊
- Matplotlib 📈
- Streamlit 🌐

---

## 📊 Features

- Dataset preview inside dashboard
- Sales analysis by category
- Profit analysis by category
- Interactive bar charts
- Automatic insights generation

---

## 📌 Key Insights

- Technology category has the highest sales
- Office Supplies has the lowest sales
- Profit does not always match sales trend
- Business insights are automatically generated
